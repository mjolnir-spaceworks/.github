# We build rockets like they should be built 🚀

MJOLNIR SPACEWORKS is developing hybrid rocket engines and weldless fuel tanks for a future where launch hardware is manufactured with speed, precision, and repeatability.

This is not only a propulsion problem. It is a manufacturing problem, a systems problem, and a software problem.

We believe the next generation of space hardware will be won by teams that can combine strong physical engineering with world-class software, automation, and production thinking.

## For Hardware Engineers ⚙️

If you want to work on real propulsion systems, design for manufacturability, test fast, and turn difficult physics into dependable products, there is room for you here.

## For Software Engineers 💻

If you want to build the tooling, controls, simulation, infrastructure, and engineering automation that make advanced hardware scalable, there is room for you here too.
